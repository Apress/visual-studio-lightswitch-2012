Extensibility projects created with the Visual Studio LightSwitch 2011 Extensibility Toolkit can be upgraded to run in Visual Studio 11 Beta by making a few modifications to the project files.

Note: Make sure that both the LightSwitch Extensibility Toolkit for Visual Studio 11 and the Visual Studio 11 SDK are installed before upgrading your projects.

Upgrade the Solution

  1. Open the solution in Visual Studio 11 and in the Review Project And Solution Changes dialog box choose OK. The solution will be upgraded and loaded; the old solution will be saved in a Backup folder.
  2. Close the solution and navigate to the directory that contains the upgraded project files.

Modify the Lspkg Project File

  1. In the ProjectName.Lspkg folder, select the ProjectName.Lspkg file and open it in Notepad.
  2. In the <PropertyGroup> section, locate the <LightSwitchVersion> property and change the value from v1.0 to v2.0.
  3. Save the file.

Modify the Client Project File

  1. In the ProjectName.Client folder, select the ProjectName.Client file and open it in Notepad.
  2. In the <PropertyGroup> section, locate the <TargetFrameworkVersion> property and change the value from v4.0 to v5.0.
  3. Locate the <LightSwitchPath Condition=" '$(LightSwitchPath)' == '' ">$(Registry:HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\VisualStudio\10.0@InstallDir)</LightSwitchPath> property and change it to <LightSwitchPath Condition=" '$(LightSwitchPath)' == '' ">$(Registry:HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\VisualStudio\11.0@InstallDir)</LightSwitchPath>.
  4. Locate the reference to System.ComponentModel.Composition and remove the metadata for Version, Culture, PublicKeyToken, and processorArchitecture. After removal the reference should be

    <Reference Include="System.ComponentModel.Composition">

              <Private>False</Private>

    </Reference>.
  5. Locate the reference to Microsoft.LightSwitch and change the HintPath from <HintPath>$(LightSwitchPath)\LightSwitch\1.0\Client\Microsoft.LightSwitch.dll</HintPath> to <HintPath>$(LightSwitchPath)\LightSwitch\Client\Microsoft.LightSwitch.dll</HintPath>.
  6. Locate the reference to Microsoft.LightSwitch.Client and change the HintPath from <HintPath>$(LightSwitchPath)\LightSwitch\1.0\Client\Microsoft.LightSwitch.Client.dll</HintPath> to <HintPath>$(LightSwitchPath)\LightSwitch\Client\Microsoft.LightSwitch.Client.dll</HintPath>.
  7. Locate the reference to Microsoft.LightSwitch.SdkProxy and change the HintPath from <HintPath>$(LightSwitchPath)\LightSwitch\1.0\Client\Microsoft.LightSwitch.SdkProxy.dll</HintPath> to <HintPath>$(LightSwitchPath)\LightSwitch\Client\Microsoft.LightSwitch.SdkProxy.dll</HintPath>.
  8. Save the file.
